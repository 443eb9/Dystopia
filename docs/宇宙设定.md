# 宇宙 Cosmos

显然在这里使用真实的宇宙比例是不现实的。因此我们将对各个数据进行缩放处理。

| ---------- | ----------------- | ---- | ----------------------- | ------------------ |

| 天文单位   | Astronomical Unit | AU   | $149,597,870,700$       |                    |
| 米         | Meter             | m    | $1$                     | 1 (1.0)            |
